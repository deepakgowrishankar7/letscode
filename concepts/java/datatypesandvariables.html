Java Data Types

In Java, data types are categorized into two main types: Primitive Data Types and Non-Primitive (Reference) Data Types. Here's a concise breakdown:

1. Primitive Data Types

These are the most basic data types in Java, predefined by the language. They store simple values.

Type Size Default Value Description
byte 1 byte (8 bits) 0 Stores small integers (-128 to 127).
short 2 bytes 0 Stores medium integers (-32,768 to 32,767).
int 4 bytes 0 Stores integers (-231 to 231-1).
long 8 bytes 0L Stores large integers (-263 to 263-1).
float 4 bytes 0.0f Stores decimal numbers (single precision).
double 8 bytes 0.0d Stores decimal numbers (double precision).
char 2 bytes '\u0000' Stores a single character (Unicode).
boolean 1 bit false Stores true or false.

2. Non-Primitive (Reference) Data Types

These are more complex and include objects, arrays, and classes. They reference memory locations rather than storing the actual data.

Examples:

  • String name = "Hello";
  • int[] numbers = {1, 2, 3};
  • MyClass obj = new MyClass();
  • interface Animal { void sound(); }

Key Differences Between Primitive and Non-Primitive Types

  • Primitive types are predefined and store values directly.
  • Non-primitive types are created by the programmer and store references to objects in memory.
  • Non-primitive types can be null, while primitive types cannot.

Java's strong typing ensures that each variable is explicitly declared with a data type, making the code robust and less error-prone.

Example Code 
    public class DataTypesExample {
    public static void main(String[] args) {
        // Primitive types
        int age = 25;
        double salary = 50000.75;
        char grade = 'A';
        boolean isJavaFun = true;

        // Non-primitive types
        String name = "John Doe";
        int[] numbers = {1, 2, 3, 4, 5};

        // Output
        System.out.println("Age: " + age);
        System.out.println("Salary: " + salary);
        System.out.println("Grade: " + grade);
        System.out.println("Is Java fun? " + isJavaFun);
        System.out.println("Name: " + name);
        System.out.println("Numbers: " + java.util.Arrays.toString(numbers));
    }
}

Java Variables

In Java, variables are containers used to store data values. They are fundamental to programming and come in different types based on the kind of data they hold. Here's a concise overview:

1. Declaring Variables

A variable in Java is declared with a specific data type, followed by its name.

int age = 25; // 'age' is a variable of type int

2. Types of Variables

a. Local Variables

  • Declared inside a method, constructor, or block.
  • Only accessible within the scope where they are defined.
  • Must be initialized before use.
void display() {
    int localVar = 10; // Local variable
    System.out.println(localVar);
}

b. Instance Variables

  • Declared inside a class but outside any method.
  • Belong to an instance of the class (each object has its own copy).
  • Default values are assigned if not explicitly initialized.
class Person {
    String name; // Instance variable
    int age;
}

c. Static Variables

  • Declared with the static keyword.
  • Shared among all instances of the class (only one copy exists).
  • Can be accessed without creating an object.
class Example {
    static int count = 0; // Static variable
}

3. Data Types

Java variables can hold different types of data:

  • Primitive Types: int, float, double, char, boolean, etc.
  • Reference Types: Objects, arrays, or instances of classes.

4. Naming Rules

  • Must start with a letter, $, or _.
  • Cannot use reserved keywords.
  • Should be meaningful and follow camelCase convention.

Example

This program demonstrates all three types of variables in action:

public class VariablesExample {
    static int staticVar = 100; // Static variable

    String instanceVar = "Hello"; // Instance variable

    public void method() {
        int localVar = 50; // Local variable
        System.out.println("Local Variable: " + localVar);
    }

    public static void main(String[] args) {
        VariablesExample obj = new VariablesExample();
        obj.method();
        System.out.println("Instance Variable: " + obj.instanceVar);
        System.out.println("Static Variable: " + staticVar);
    }
}
