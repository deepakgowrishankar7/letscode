Java Strings

In Java, String is a class that represents a sequence of characters. Strings are immutable, meaning once created, their values cannot be changed.

1. Creating Strings

  • Using String Literal: String s = "Hello";
  • Using new Keyword: String s = new String("Hello");
public class StringCreate {
    public static void main(String[] args) {
        String s1 = "Hello";
        String s2 = new String("World");
        System.out.println(s1);
        System.out.println(s2);
    }
}
Output: 
Hello
World

2. Common String Methods

  • length() – Returns the number of characters.
  • charAt(index) – Returns character at given index.
  • toUpperCase() / toLowerCase() – Changes case.
  • equals() / equalsIgnoreCase() – Compares two strings.
  • contains() – Checks if a string contains a sequence.
  • replace() – Replaces characters or text.
  • substring() – Extracts a portion.
  • trim() – Removes leading/trailing spaces.
  • split() – Splits the string into array.
  • concat() – Appends another string.
public class StringMethods {
    public static void main(String[] args) {
        String str = "  Java Programming ";

        System.out.println("Length: " + str.length());
        System.out.println("Char at index 5: " + str.charAt(5));
        System.out.println("Uppercase: " + str.toUpperCase());
        System.out.println("Lowercase: " + str.toLowerCase());
        System.out.println("Trimmed: " + str.trim());
        System.out.println("Contains 'Pro': " + str.contains("Pro"));
        System.out.println("Replaced: " + str.replace("Java", "Python"));
    }
}
Output: 
Length: 19
Char at index 5: P
Uppercase:   JAVA PROGRAMMING 
Lowercase:   java programming 
Trimmed: Java Programming
Contains 'Pro': true
Replaced:   Python Programming

3. Comparing Strings

public class CompareStrings {
    public static void main(String[] args) {
        String a = "hello";
        String b = "HELLO";

        System.out.println(a.equals(b));
        System.out.println(a.equalsIgnoreCase(b));
    }
}
Output: 
false
true

4. String Concatenation

public class ConcatExample {
    public static void main(String[] args) {
        String first = "Java";
        String second = "Language";

        System.out.println(first + " " + second);
        System.out.println(first.concat(" " + second));
    }
}
Output: 
Java Language
Java Language

5. Splitting Strings

public class SplitExample {
    public static void main(String[] args) {
        String sentence = "Java is simple";
        String[] words = sentence.split(" ");

        for (String word : words) {
            System.out.println(word);
        }
    }
}
Output: 
Java
is
simple

6. Using substring()

public class SubstringExample {
    public static void main(String[] args) {
        String s = "Programming";
        System.out.println(s.substring(0, 4)); // from 0 to 3
    }
}
Output: 
Prog

7. StringBuilder and StringBuffer

  • StringBuilder: Mutable, faster, not thread-safe
  • StringBuffer: Mutable, slower, but thread-safe

Example (StringBuilder):

public class BuilderExample {
    public static void main(String[] args) {
        StringBuilder sb = new StringBuilder("Hello");
        sb.append(" World");
        System.out.println(sb.toString());
    }
}
Output: 
Hello World

Example (StringBuffer):

public class BufferExample {
    public static void main(String[] args) {
        StringBuffer sb = new StringBuffer("Java");
        sb.append(" Strings");
        System.out.println(sb);
    }
}
Output: 
Java Strings
