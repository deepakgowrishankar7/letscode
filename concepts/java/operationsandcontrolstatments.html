Java Operations and Control Statements

Java provides various operators and control statements to manipulate data and control the flow of the program.

Types of Operators:

Java operators are used to perform operations on variables and values. Here's a beginner-friendly overview with examples and outputs.

  • Arithmetic Operators: +, -, *, /, %
  • Relational Operators: ==, !=, >, <, >=, <=
  • Logical Operators: &&, ||, !
  • Assignment Operators: =, +=, -=, *=, /=

1. Arithmetic Operators

Used for basic math operations:

  • + : Addition
  • - : Subtraction
  • * : Multiplication
  • / : Division
  • % : Modulus (Remainder)
Example Code: 
public class ArithmeticExample {
    public static void main(String[] args) {
        int a = 10, b = 3;
        System.out.println("a + b = " + (a + b));
        System.out.println("a - b = " + (a - b));
        System.out.println("a * b = " + (a * b));
        System.out.println("a / b = " + (a / b));
        System.out.println("a % b = " + (a % b));
    }
}
Expected Output: 
a + b = 13
a - b = 7
a * b = 30
a / b = 3
a % b = 1

2. Relational Operators

Used to compare values:

  • == : Equal to
  • != : Not equal to
  • > : Greater than
  • < : Less than
  • >= : Greater than or equal to
  • <= : Less than or equal to
Example Code: 
public class RelationalExample {
    public static void main(String[] args) {
        int x = 5, y = 10;
        System.out.println(x == y);
        System.out.println(x != y);
        System.out.println(x > y);
        System.out.println(x < y);
        System.out.println(x >= 5);
        System.out.println(y <= 5);
    }
}
Expected Output: 
false
true
false
true
true
false

3. Logical Operators

Used to combine boolean expressions:

  • && : Logical AND
  • || : Logical OR
  • ! : Logical NOT
Example Code: 
public class LogicalExample {
    public static void main(String[] args) {
        int a = 7, b = 12;
        System.out.println(a > 5 && b > 10);
        System.out.println(a < 5 || b > 10);
        System.out.println(!(a == b));
    }
}
Expected Output: 
true
true
true

4. Assignment Operators

Used to assign values to variables:

  • = : Assign
  • += : Add and assign
  • -= : Subtract and assign
  • *= : Multiply and assign
  • /= : Divide and assign
Example Code: 
public class AssignmentExample {
    public static void main(String[] args) {
        int a = 10;
        System.out.println("Original a = " + a);
        a += 5;
        System.out.println("After += 5, a = " + a);
        a -= 3;
        System.out.println("After -= 3, a = " + a);
        a *= 2;
        System.out.println("After *= 2, a = " + a);
        a /= 4;
        System.out.println("After /= 4, a = " + a);
    }
}
Expected Output: 
Original a = 10
After += 5, a = 15
After -= 3, a = 12
After *= 2, a = 24
After /= 4, a = 6

Java Conditional Statements:

  • Conditional Statements: if, if-else, else-if ladder, switch
  • Looping Statements: for, while, do-while
  • Jump Statements: break, continue, return

    • Even or Odd using If-Else:

    import java.util.Scanner;
public class EvenOddCheck {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    System.out.println("Enter a number:");
    int num = sc.nextInt();
    if (num % 2 == 0) {
        System.out.println("Even number");
    } else {
        System.out.println("Odd number");
    }
    sc.close();
  }
}

    Grading System using Else-If Ladder:

    import java.util.Scanner;
public class GradeCheck {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    System.out.println("Enter your percentage:");
    int percentage = sc.nextInt();

    if (percentage > 85)
        System.out.println("First Class");
    else if (percentage > 70)
        System.out.println("Second Class");
    else if (percentage > 35)
        System.out.println("Third Class");
    else
        System.out.println("Fail");
    
    sc.close();
  }
}

    Day Type using Switch Case:

    import java.util.Scanner;
public class DayType {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    System.out.println("Enter a day:");
    String day = sc.next().toLowerCase();

    switch (day) {
      case "monday":
      case "tuesday":
      case "wednesday":
      case "thursday":
      case "friday":
        System.out.println("Weekday");
        break;
      case "saturday":
      case "sunday":
        System.out.println("Weekend");
        break;
      default:
        System.out.println("Invalid day");
    }
    sc.close();
  }
}

    Java Looping Statements:

    For Loop - Print Numbers 1 to 5:

    public class ForLoopExample {
  public static void main(String[] args) {
    for (int i = 1; i <= 5; i++) {
        System.out.println(i);
    }
  }
}

    While Loop - Sum of Digits:

    import java.util.Scanner;
public class SumDigits {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    System.out.println("Enter number:");
    int num = sc.nextInt();
    int sum = 0;
    while (num != 0) {
      sum += num % 10;
      num /= 10;
    }
    System.out.println("Sum of digits: " + sum);
    sc.close();
  }
}

    Do-While Loop - Repeat until user enters 0:

    import java.util.Scanner;
public class DoWhileExample {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    int num;
    do {
      System.out.println("Enter a number (0 to stop):");
      num = sc.nextInt();
    } while (num != 0);
    System.out.println("Exited loop.");
    sc.close();
  }
}

    Java Jump Statements:

    Break Statement - Stop loop at 5:

    public class BreakExample {
  public static void main(String[] args) {
    for (int i = 1; i <= 10; i++) {
      if (i == 5) {
        break;
      }
      System.out.println(i);
    }
  }
}

    Continue Statement - Skip even numbers:

    public class ContinueExample {
  public static void main(String[] args) {
    for (int i = 1; i <= 10; i++) {
      if (i % 2 == 0) {
        continue;
      }
      System.out.println(i);
    }
  }
}

    Return Statement - Exit from a method early:

    public class ReturnExample {
  public static void check(int num) {
    if (num < 0) {
      System.out.println("Negative number");
      return;
    }
    System.out.println("Positive number");
  }

  public static void main(String[] args) {
    check(10);
    check(-5);
  }
}
