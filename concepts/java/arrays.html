Java Arrays

An array in Java is a container object that holds a fixed number of values of a single type. It's used when you want to store multiple values in one variable instead of declaring separate variables for each value.

  • Definition: An array is a fixed-length, indexed collection of elements of the same data type.
  • Why Arrays? Efficient to group related data together and process them using loops.
  • Syntax: datatype[] arrayName; or datatype arrayName[];
  • Declaration + Initialization (Static): int[] marks = {85, 90, 78};
  • Declaration + Initialization (Dynamic): int[] marks = new int[3]; → creates an array with default values
  • Access Elements: marks[0], marks[1]...
  • Update Elements: marks[2] = 100;
  • Array Length: marks.length gives number of elements
  • Default Values:
    • int, byte, short, long → 0
    • float, double → 0.0
    • boolean → false
    • char → '\u0000'
    • Object types (String, etc) → null
  • Looping through Arrays: Use for, while, or for-each
  • Enhanced For Loop: for(int mark : marks) { System.out.println(mark); }
  • Two-Dimensional Array: int[][] matrix = new int[2][3];
  • Jagged Arrays: 2D arrays with varying column sizes: new int[3][];
  • ArrayIndexOutOfBoundsException: Runtime error if accessing an invalid index
  • Array Cloning: int[] copy = marks.clone();
  • Sorting Arrays: Arrays.sort(marks); from java.util package
  • Printing Arrays: System.out.println(Arrays.toString(marks));
  • When to Use: Use arrays when data size is known and fixed (e.g., student marks, prices, IDs)

Example 1: One-Dimensional Array


import java.util.Arrays;

public class ArrayExample {
    public static void main(String[] args) {
        int[] scores = {75, 80, 90};

        System.out.println("First Score: " + scores[0]);
        System.out.println("All Scores: " + Arrays.toString(scores));

        // Loop through array
        for (int i = 0; i < scores.length; i++) {
            System.out.println("Score at index " + i + ": " + scores[i]);
        }
    }
}
Output: 

First Score: 75
All Scores: [75, 80, 90]
Score at index 0: 75
Score at index 1: 80
Score at index 2: 90

Example 2: Two-Dimensional Array


public class TwoDArrayExample {
    public static void main(String[] args) {
        int[][] matrix = {
            {1, 2},
            {3, 4}
        };

        for (int i = 0; i < matrix.length; i++) {
            for (int j = 0; j < matrix[i].length; j++) {
                System.out.print(matrix[i][j] + " ");
            }
            System.out.println();
        }
    }
}
Output: 

1 2
3 4

Common Error: Index Out of Bounds


public class ErrorExample {
    public static void main(String[] args) {
        int[] arr = new int[3];
        System.out.println(arr[3]); // Error: Index 3 is out of bounds for length 3
    }
}

Array Operations

  • Accessing Elements: arr[index]
  • Updating Elements: arr[index] = value;
  • Finding Length: arr.length
  • Iterating with Loops: for, for-each

Why Use Arrays?

  • Efficient way to handle large amounts of data
  • Useful for loops and collections
  • Foundational structure for more complex data types
