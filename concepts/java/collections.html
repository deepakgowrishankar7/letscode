Java Collections Framework

The Java Collections Framework is a set of classes and interfaces that implement commonly reusable data structures like List, Set, Map, and Queue. It provides a way to store, access, and manipulate groups of objects efficiently.

Main Interfaces in Collections:

  • List – Ordered, allows duplicates (e.g., ArrayList, LinkedList)
  • Set – Unordered, no duplicates (e.g., HashSet, TreeSet)
  • Map – Key-value pairs (e.g., HashMap, TreeMap)
  • Queue – FIFO order (e.g., LinkedList, PriorityQueue)

1. List Example (ArrayList)

List stores elements in the order they were added and allows duplicates.

import java.util.ArrayList;

public class ListExample {
    public static void main(String[] args) {
        ArrayList fruits = new ArrayList<>();
        fruits.add("Apple");
        fruits.add("Banana");
        fruits.add("Mango");

        System.out.println(fruits);
    }
}
Output: 
[Apple, Banana, Mango]

2. Set Example (HashSet)

Set does not allow duplicate values and does not maintain insertion order.

import java.util.HashSet;

public class SetExample {
    public static void main(String[] args) {
        HashSet colors = new HashSet<>();
        colors.add("Red");
        colors.add("Green");
        colors.add("Red"); // duplicate

        System.out.println(colors);
    }
}
Output: 
[Red, Green]

3. Map Example (HashMap)

Map stores key-value pairs. Keys are unique.

import java.util.HashMap;

public class MapExample {
    public static void main(String[] args) {
        HashMap map = new HashMap<>();
        map.put(1, "A");
        map.put(2, "B");
        map.put(3, "C");

        System.out.println(map);
    }
}
Output: 
{1=A, 2=B, 3=C}

4. Queue Example (LinkedList)

Queue follows FIFO (First In First Out).

import java.util.LinkedList;
import java.util.Queue;

public class QueueExample {
    public static void main(String[] args) {
        Queue tasks = new LinkedList<>();
        tasks.add("Task1");
        tasks.add("Task2");

        System.out.println("Next: " + tasks.peek());
        tasks.remove();
        System.out.println("After removing: " + tasks);
    }
}
Output: 
Next: Task1
After removing: [Task2]

5. Iterating Collections (For-each loop)

import java.util.ArrayList;

public class LoopExample {
    public static void main(String[] args) {
        ArrayList nums = new ArrayList<>();
        nums.add(10);
        nums.add(20);
        nums.add(30);

        for (int n : nums) {
            System.out.println(n);
        }
    }
}
Output: 
10
20
30

6. Useful Methods in Collection Classes

  • add() – Add an element
  • remove() – Remove an element
  • contains() – Check if element exists
  • size() – Get size of collection
  • clear() – Remove all elements
  • isEmpty() – Check if collection is empty

7. Sorting Collections

import java.util.ArrayList;
import java.util.Collections;

public class SortExample {
    public static void main(String[] args) {
        ArrayList names = new ArrayList<>();
        names.add("Zara");
        names.add("Anil");
        names.add("John");

        Collections.sort(names);
        System.out.println(names);
    }
}
Output: 
[Anil, John, Zara]

8. Convert Array to List

import java.util.Arrays;
import java.util.List;

public class ConvertArrayToList {
    public static void main(String[] args) {
        String[] animals = {"Dog", "Cat", "Horse"};
        List list = Arrays.asList(animals);

        System.out.println(list);
    }
}
Output: 
[Dog, Cat, Horse]
