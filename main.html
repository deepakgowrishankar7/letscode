Java Introduction

Java is a high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible. It was originally developed by James Gosling at Sun Microsystems and released in 1995.

Features of Java

  • Object-Oriented
  • Platform Independent (via the Java Virtual Machine)
  • Simple and Secure
  • Multithreaded and Dynamic
  • High Performance through Just-In-Time Compiler

Hello World Example

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Java Development Kit (JDK)

The JDK includes tools for developing and testing Java programs, including the compiler (javac), the interpreter (java), and more.

JVM and Bytecode

Java source code is compiled into bytecode, which runs on the Java Virtual Machine (JVM). This allows Java programs to be platform-independent.

Basic Syntax

  • Case-sensitive
  • Class names should start with uppercase letters
  • Main method signature: public static void main(String[] args)

Data Types

Java has two types of data types: primitive (int, boolean, etc.) and non-primitive (arrays, classes, etc.).

Control Statements

Java includes if-else, switch, loops (for, while, do-while), and branching statements like break and continue.

Object-Oriented Concepts

  • Class and Object
  • Inheritance
  • Encapsulation
  • Polymorphism
  • Abstraction

Conclusion

Java is a powerful, versatile language used in various domains like web development, mobile applications (Android), enterprise solutions, and more. Its robustness and platform independence make it a popular choice among developers worldwide.
